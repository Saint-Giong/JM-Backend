# JM-Backend
This is the SaintGiong-JobManager Backend repository which contains sub-modules

## Prerequisites
- Java 17+
- Maven 3.8\+
- GitHub account with access to repo `'Saint-Giong/SG-SharedDtoPackage'`

## 1\. Installation
- Clone the repository
```xml aiignore
git clone https://github.com/Saint-Giong/JM-Backend.git
```

## 2\. Repository Fetching
- Fetch latest changes to `submodules`
```xml aiignore
git submodule update --remote
```

- If you have `nested submodules`, include the `--recursive` flag
```xml
git submodule update --remote --recursive
```
